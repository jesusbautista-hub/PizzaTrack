# 🍕 Proyecto Pizza-Track

Este proyecto es una aplicación de consola en Java que implementa una estructura de datos de **Pila (Stack)** utilizando memoria dinámica (nodos y punteros).

## 🚀 Funcionalidades
El programa simula el sistema de pedidos de una pizzería, permitiendo:
1. **Push:** Agregar un nuevo pedido de pizza (con un arreglo de 3 ingredientes).
2. **Pop:** Deshacer el último pedido realizado (LIFO - Last In, First Out).
3. **Mostrar:** Ver el historial completo de pedidos apilados.

## 🛠️ Tecnologías Usadas
* Java (Eclipse Temurin)
* Visual Studio Code
* Git y GitHub para el control de versiones
